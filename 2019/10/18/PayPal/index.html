关于https://www.paypal.com/sdk/js需要传递的参数有两个（在官网上看到的解释）：

Option Example value Default Description
client-id sb or 你申请到的client-id required PayPal REST客户端ID。用于识别PayPal帐户并确定在何处付款。在沙盒中测试时，可以将client-id = sb用作快捷方式。
currency AUD USD USD 收款的默认货币

如果你需要用澳元的话那就是：https://www.paypal.com/sdk/js?client-id=sb&currency=AUD
